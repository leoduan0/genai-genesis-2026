# Screening Taxonomy

## Stage One Dimensions (8 spectra)

1. Distress
2. Fear
3. Disinhibited Externalizing
4. Antagonistic Externalizing
5. Thought Disorder
6. Detachment
7. Somatoform
8. Compulsivity

## Stage Two Conditions by Spectrum

### Distress
- Major depressive disorder
- Persistent depressive disorder (dysthymia)
- Generalized anxiety disorder
- PTSD
- Adjustment disorder (depressive/anxious type)

### Fear
- Panic disorder
- Agoraphobia
- Social anxiety disorder
- Specific phobias
- Separation anxiety disorder

### Disinhibited Externalizing
- ADHD
- Alcohol use disorder
- Drug use disorder
- Gambling / behavioral addictions
- Antisocial behavior / conduct problems
- Binge eating disorder

### Antagonistic Externalizing
- Narcissistic features
- Borderline features (antagonistic component)

### Thought Disorder
- Schizophrenia-spectrum (hallucinations, delusions)
- Bipolar I (manic/psychotic features)
- Bipolar II (hypomanic features)
- Schizotypal features
- Attenuated psychosis / clinical high risk

### Detachment
- Schizoid features
- Avoidant features
- Depersonalization / derealization
- Autism-spectrum features

### Somatoform
- Somatic symptom disorder
- Illness anxiety disorder

### Compulsivity
- OCD
- Body dysmorphic disorder
- Hoarding disorder
- Anorexia nervosa
- OCPD features
- Tic disorders

## Totals

8 spectra, 35 conditions

## Cross-loading notes

Some conditions load meaningfully on multiple spectra. Primary placement is listed above; significant secondary loadings:

- PTSD: distress (primary) + fear (re-experiencing/avoidance)
- GAD: distress (primary) + fear (somatic anxiety component)
- Bipolar I: thought disorder (primary) + distress (depressive pole)
- Bipolar II: thought disorder (primary) + distress (depressive pole, stronger than BP-I)
- Borderline: antagonistic (primary) + distress (emotional dysregulation)
- Antisocial: disinhibited (primary) + antagonistic (interpersonal)
- BDD: compulsivity (primary) + somatoform (body focus)
- Anorexia: compulsivity (primary) + somatoform (body focus) + detachment (emotional restriction)
- Binge eating: disinhibited (primary) + distress (emotional eating)
- Avoidant features: detachment (primary) + fear (social evaluative)
- Autism-spectrum: detachment (primary) + compulsivity (rigidity, restricted interests)
- Illness anxiety: somatoform (primary) + fear (health-specific phobia)
- Adjustment disorder: distress (primary) — weak loadings everywhere, essentially a subthreshold p-factor elevation
